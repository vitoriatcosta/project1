Aula Doctype + Meta Tags

Doctype is a instruction for browser interpratation.

Meta tags provide more informations to browsers about our webpage, what are we looking at. Meta Data = additional data. These tags go into the header, as the character set; SEO for search engines just as description and keywords.The Meta Author tag contains the name of the person coding the webpage.


Responsive web design

Viewport Meta tag is very important for responsive web design, which means the web design has to "look good" on every device and tab size used to acess the web page. Bootstrap is commonly used as a framework for this matter. In order to use Bootstrap, the viewport meta tag must be used
